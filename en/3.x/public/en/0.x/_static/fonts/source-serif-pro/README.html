Source Serif Pro¶

Source Serif Pro is a set of OpenType fonts to complement the Source Sans Pro family. In addition to a functional OpenType font, this open source project provides all of the source files that were used to build this OpenType font by using the AFDKO makeotf tool.

Installation instructions¶

  • Mac OS X
  • Windows
  • Linux/Unix-based systems

Getting Involved¶

Send suggestions for changes to the Source Serif OpenType font project maintainer, [Frank Grießhammer](mailto:opensourcefonts@adobe.com?subject=[GitHub] Source Serif Pro), for consideration.

Further information¶

For information about the design and background of Source Serif, please refer to the official font readme file.
